Shapes

Our basic building blocks.

Standard parameters: position, rotation

Sphere

Parameters: radius

Box

Parameters: size (x, y, z)

RoundBox

Parameters: size (x, y, z), corner radius

Cylinder

Parameters: radius, height

Torus

Parameters: radius, thickness

Union

union(sphere, sphere) = butt

Parameters: none

Subtract

subtract(sphere, sphere)

Parameters: none

Intersect

intersect(sphere, sphere)

The shape where two other shapes overlap.

Parameters: none

Smooth

smooth(sphere, sphere, 0.2)

Same as Union, but rounds off the join.

Parameters: amount

Blend

blend(sphere, box, 0.5)

Blends smoothly between two primitives.

Parameters: amount

Noise

sphere + 1D noise sphere + 2D noise sphere + 3D noise

Random permutation of the surface. Super important and versatile, our swiss army knife.

Parameters: amount, scale (x, y, z), speed

Wave

box + wave

parameters: amplitude, frequency, axis

Twist

box + twist(0.5) box + twist(3)

parameters: amount, axis

Repeat

sphere + repeat

Repeats a primitive to infinity.

Not very blob by itself, but can be cropped with a Subtract or Intersect.

intersect(box + repeat, sphere)

Parameters: spacing (x, y, z)
